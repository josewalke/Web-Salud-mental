Google Search Console Verification

Este archivo es para verificar tu sitio en Google Search Console.

Instrucciones:

  1. Ve a Google Search Console
  2. Agrega tu propiedad (dominio)
  3. Elige "Archivo HTML" como método de verificación
  4. Copia el código de verificación
  5. Reemplaza "REEMPLAZA_CON_TU_CODIGO" en este archivo
  6. Sube el archivo a tu servidor
  7. Verifica tu propiedad
  8. Envía tu sitemap.xml

Después de la verificación:

  • Envía tu sitemap.xml
  • Monitorea el rendimiento de búsqueda
  • Revisa y corrige errores de indexación
  • Optimiza para Core Web Vitals
